# Seed Repository

This repository provides a structured dataset for populating your database with official Ukrainian administrative and settlement data.

The data is sourced from the national register of territorial units and organized into a normalized hierarchical structure, making it easy to integrate into applications that require geographical or administrative information.

## Features

* Official location data based on the national registry
* Hierarchical administrative structure
* Type-safe TypeScript datasets
* Ready for database seeding
* Consistent codes and parent-child relationships

## Data Structure

The repository groups locations by administrative categories, preserving their relationships and unique identifiers. This allows applications to efficiently query and navigate the hierarchy, from regions down to individual settlements.
